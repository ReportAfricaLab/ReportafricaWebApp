## How to Create Courses

1. Go to: reportafrica-web-app-admin.vercel.app/courses
2. Login with your admin account
3. Click "+ Add Course"
4. Fill in the details below
5. Click "Save"
6. Click "+ Add Lesson" for each lesson
7. Check "Published" when ready to go live

## Video URL Format

When you upload videos to YouTube (as Unlisted), the embed URL format is:

```
https://www.youtube.com/embed/VIDEO_ID
```

Example: If your YouTube video URL is `https://www.youtube.com/watch?v=abc123`
Then the embed URL is: `https://www.youtube.com/embed/abc123`

You can also use Vimeo: `https://player.vimeo.com/video/VIDEO_ID`
